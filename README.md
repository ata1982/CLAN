# NextJS Project

Apple風デザインシステムを採用したNext.jsプロジェクトです。

## 開発開始

```bash
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## ビルド & デプロイ

```bash
# 開発
npm run dev

# ビルド
npm run build

# 本番確認
npm run start

# テスト
npm run test

# リント
npm run lint
```

## 技術スタック

- **Framework**: Next.js 15
- **Styling**: TailwindCSS 4 + Apple Design System
- **TypeScript**: フル対応
- **Testing**: Jest + Testing Library
